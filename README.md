## 🟢 주요 기능
- 회원가입, 로그인
- 카카오, 네이버 로그인
- 마이페이지
- 사용자 점수로 회원 등급 표시
- 게시글 작성, 수정, 삭제, 보기
- 게시글 조회수, 찜, 싫어요, 댓글 작성, 댓글 삭제
- 네이버 지도 API를 활용한 지역 정보 및 길찾기 기능

## 🟡 기술 스택
- React.js
- TypeScript
- React Router
- React Cookie
- Zustand
- Axios
- React Daum Postcode

## 🟣 프로젝트 실행 방법
### 클론 및 폴더 이동
```bash
git clone 깃주소
cd 프로젝트명
```

### 패키지 설치
```bash
npm install 
```

### 개발 서버 실행
```bash
npm run start
```

### 빌드 방법
```bash
npm run build
```
